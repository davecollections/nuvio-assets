# Manual source artwork

These files are owner-approved render inputs for narrowly scoped studio/network source treatments. They do not replace or erase TMDB source history. Each treatment remains bound to its stable key, canonical name, current TMDB logo path, original TMDB source hash, selected source hash, and production background in `config/source-treatments.json`.

The artwork is retained for Nuvio cover generation with provenance, integrity, and invalidation metadata. Logo and trademark rights may apply; inclusion here is not a claim that the marks are freely licensed or available for unrelated reuse.

## Included sources

- `companies/1742/les-films-13-official.png` — Les Films 13 wordmark from the organisation's official website. Source page: <https://www.lesfilms13.com/>.
- `companies/80138/cpb-films-official.png` — CPB Films mark from the current official Incognita Studio page. Source page: <https://www.incognita.fr/cpb-films/>.

No fan-made, watermarked, or unclear-provenance artwork is included.
